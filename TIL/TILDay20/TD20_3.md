### 2차원 배열 (2020.8.15) - 3

#### 2차원 배열의 예시

```java
// Case 1. 
//	Int [ ][ ] array1 = new int [3][4];

//	array1 -> [    ][    ][    ][    ]
//	        -> [    ][    ][    ][    ]
//		-> [    ][    ][    ][    ]
	
//	array1 [0][1] = 10;
//	array1 -> [    ][ 10 ][    ][    ]
//		-> [    ][    ][    ][    ]
//		-> [    ][    ][    ][    ]
```


```java
// Case 2.
```
